# teachio-landing

Landing page for presenting a bachelor thesis project.

## Stage 1: Initial Setup

Implemented in this stage:

- Project base structure with semantic HTML page.
- CSS reset and baseline custom styles.
- Basic SEO metadata and social tags.
- Search crawler files: `robots.txt` and `sitemap.xml`.
- Favicon set for multiple devices and sizes.
